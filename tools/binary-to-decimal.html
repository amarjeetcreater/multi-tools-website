Number Systems Reference

Binary (Base-2)
  • Uses only 0 and 1
  • Each position represents a power of 2
  • Common in computer systems
  • Examples: 1010₂ = 10₁₀
  • 1111₂ = 15₁₀
Decimal (Base-10)
  • Uses digits 0-9
  • Each position represents a power of 10
  • Most common number system
  • Examples: 10₁₀ = 1010₂
  • 15₁₀ = 1111₂
