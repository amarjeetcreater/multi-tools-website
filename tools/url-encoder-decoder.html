URL Encoder & Decoder

Encode and decode URLs for web compatibility

About URL Encoding

This tool handles different types of encoding:

  • URL Encoding: Converts special characters to %XX format
  • Component Encoding: Encodes URL components
  • Base64: Converts to Base64 format
  • HTML Entities: Converts to HTML entities
Examples
URL: https://example.com/path with spaces
→ https%3A%2F%2Fexample.com%2Fpath%20with%20spaces
HTML: <script>alert('test')</script>
→ &lt;script&gt;alert('test')&lt;/script&gt;
